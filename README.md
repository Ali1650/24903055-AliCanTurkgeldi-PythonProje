 Kişisel Rehber ve Alışveriş Listesi

Proje Hakkında:
Bu projede Python Programlama II dersi kapsamında basit bir rehber ve alışveriş listesi uygulaması yapılmıştır.  
Program sayesinde kullanıcı kişiler ekleyebilir ve alışveriş listesi oluşturabilir.

# Yapılan İşlemler
- Rehbere kişi ekleme
- Kişileri listeleme
- Kişi arama
- Kişi silme
- Alışveriş listesine ürün ekleme
- Ürünleri listeleme

# Nasıl Çalışır?
Program çalıştırıldığında ekrana bir menü gelir.  
Kullanıcı bu menüden seçim yaparak istediği işlemi gerçekleştirebilir.

 Kullanılan Yapılar
Bu projede:
- Listeler
- ChatGpt
- Fonksiyonlar  
kullanılmıştır.
